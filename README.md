# GarageWithGenerics
## Om projektet
En consoleapplikation som hanterar ett garage.
Applikationen använder sig mest av listor men arrayer förekomer.

### Funktionalitet
1. Man kan lägga till olika fordon.
2. Går att lista alla fordon och fordons typer.
3. Ta bort fordon.
4. Söka efter fordon specifikt kopplat till registreringsnummer eller färg.
5. Finns även en mer avancerad sökmetod som hanterar flera sökord för att hitta ett specifikt fordon.
6. Man kan även bygga ut garaget med en plats ifall det är fullt!
7. Sen går det att stänga ner garaget.

### Användbarhet
Applikationen har input checks som gör att det ej ska krasha om man skriver in fel.
Samt så handhåller det användaren med att ge information om vad applikationen begär av användaren för att komma vidare.
